MediaX — Gallery

Local-first image/video viewer with tagging, search, and non-destructive edits.

Context

Large local libraries need fast triage and organization without cloud lock-in.

Problem

High-latency previews and slow metadata kill flow when sifting thousands of assets.

Solution

Rust core, efficient metadata store, cached previews, and GPU acceleration where it matters.

Results

Responsive browsing at scale, predictable storage, private by default.

Tech

Rust, SQLite/Tantivy, ONNX Runtime, optional CUDA/TensorRT, local caching.

Impact

Handled 100K+ assets with sub-second loads.
