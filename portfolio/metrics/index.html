System Metrics Logger

High-throughput metrics collector with SQLite + zstd archives.

Context

Long-term metrics without a complex TSDB or vendor lock-in.

Problem

High-write rates and storage bloat in lightweight environments.

Solution

Append-only segments, periodic compaction, zstd archives, and simple queries.

Results

Predictable footprint with good query performance on modest hardware.

Tech

Rust, SQLite, zstd, export to CSV/Parquet.

Impact

Handled 1M+ metrics/sec with low resource usage.
